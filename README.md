# TODO REACT APP

A Todo app built using ReactJs

## How to run it

In the project directory, you can run:

### `npx create-react-app todolist`

### `npm start`

## FEATURES
    - Add todos
    - Delete todos
    - Edit todos
    - Checkboxes for todos
    - Clear All button
    - Save todos to local storage and load them when site is reloaded
